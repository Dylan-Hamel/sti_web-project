# Deployment / usage

Run Docker machine, then :

```bash
$ git clone -b secured https://github.com/Dylan-Hamel/sti_web-project.git
$ cd sti_web-project/deployment/
$ ./run
```

That's it ! You may reach our app at **http://localhost/**
